# data-mining-pipeline
Preview

This repository contains the Semester 4 Data Mining Pipeline team project, demonstrating a structured end-to-end workflow for collecting, cleaning, analyzing, and visualizing datasets across multiple domains.

The project covers five key tasks, each following a modular and reproducible pipeline:

Job Market Intelligence
Scraping Polish IT job listings to analyze salary ranges, city distribution, and in-demand skills.
Machine Learning: Random Forest Regressor for salary prediction.
Media Headline Insights
Collecting and analyzing global news headlines for frequency trends and topical insights.
NLP preprocessing, word frequency analysis, and visualization.
E-Commerce Price Monitor
Tracking book prices over multiple sessions and forecasting future trends.
ML: Linear Regression with Polynomial Features for Session 6 price prediction.
Weather Trend Explorer
Retrieving and analyzing historical weather data for 10 Polish cities.
Trend analysis for temperature, precipitation, and weather conditions.
Public Health Dashboard
Aggregating global COVID-19 data and classifying countries by mortality risk tiers.
ML: Decision Tree & Logistic Regression classification models, interactive dashboards.
